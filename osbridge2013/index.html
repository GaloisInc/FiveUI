FiveUI is here to help!

We've built a tool called FiveUI

With FiveUI, you can:

  • encode guidelines
  • analyse your interfaces using the guidelines you've encoded
  • automate the analysis for your entire website
  • share your encoded guidelines with the community

FiveUI is...

  • a web browser extension (Firefox and Chrome)
  • a headless system for testing automation
  • open source (Apache 2.0)
  • at http://galoisinc.github.io/FiveUI

Getting Involved

Writing Rules

Pick your favorite UI guideline and encode it!

  • The only requirement is some knowledge of Javascript (familiarity with jQuery is a plus!)
  • Rules are written in Javascript and use the CommonJS module format
  • There are many examples in our github repo

Example:

exports.name = "imagesAltText";

exports.description = "Each image should have an alternative text description";

exports.rule = function(report) {
  fiveui.query('img')
     .filter(function (i) {
               var altAttr = $(this).attr('alt');
               return altAttr == undefined || altAttr  == '';
             })
     .each(function (i, e) {
             report.error('Image has no alt text', e);
           });
};

Contribute

  • Once you've written a useful rule, add it to the growing collection on github
  • Write general-purpose rules that apply to any website
  • The community can take advantage of the rules contributed by everyone else
