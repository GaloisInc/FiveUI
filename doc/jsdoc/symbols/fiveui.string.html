<static> {!boolean} fiveui.string.isTitleCase(str)
A simple heuristic check to see if a string is in Title Case. This does not perform an exhaustive gramatical analysis, and as such, it is prone to generating false-positives in some cases. In particular, it only has a short 'white list' of articles, conjections, and prepositions that are allowed to be in lower case.
Parameters:
{!string} str
The string to check.
Returns:
{!boolean} true if the string is in title case, false if it is not.
